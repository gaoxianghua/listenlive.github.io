## Hugo基本命令
```
hugo server --noHTTPCache					#启动
hugo server --disableFastRender				#FastRender启动
hugo new site "listenlive.cn"				#创建站点
hugo new post/note/xx.md					#创建文章
hugo 										#编译所有文件并输出到public目录
```
